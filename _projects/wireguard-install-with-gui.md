A streamlined Bash-based tool that automates the installation and configuration of [WireGuard VPN](https://www.wireguard.com/) on Linux systems, with a graphical interface powered by [wireguard-ui](https://github.com/ngoduykhanh/wireguard-ui).

Unlike many manual or command-line-only setups, this script is ideal for:
- Home users who want an easy VPN setup.
- Sysadmins needing quick deployments.
- Anyone looking for a lightweight GUI alternative for WireGuard.
